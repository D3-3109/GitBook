#### **代理模块：**

* 支持出站传输协议 <mark style="color:blue;">VLESS XTLS</mark> / <mark style="color:blue;">Trojan XTLS</mark>
* 支持出站传输协议 Http的headers功能
* 允许provider请求过clash
* Proxy-Providers支持直接使用V2rayN等客户端的<mark style="color:blue;">普通订阅</mark>
